
Irsa's Kitchen - Offline Flutter App (Prepared package)
------------------------------------

Files included:
- lib/main.dart
- assets/menu.json (cleaned from your Excel)
- BuildAPK.bat (one-click builder for Windows)
- README.txt (this file)

How to build APK (Windows):
1. Ensure Flutter is installed and on PATH.
2. Open terminal in this folder.
3. Run: flutter pub get
4. Run: flutter build apk --release
5. Find APK: build/app/outputs/flutter-apk/app-release.apk
